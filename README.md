# SSDB
### Simple simple DB
## phase 1
supports list
- two operation (insert / select)
- store only in memory (no persistence to disk)
- support single hard-code table 
## Table

For first phase the table structure are hard-code and its single one.

| Column | Type |
| ------ | ------ |
| id | integer |
| username | varchar(32) |
| email | varchar(255) |
